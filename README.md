# WELCOME

This is a very dumb but fun project. It's an app to take in coffee orders for a small, at-home coffee pop-up I'm throwing for my birthday :D

This project uses Next.js for both the frontend and API layers with React Hook Form and Zod for validation, along with a PostgreSQL DB to store the data (overkill? maybe. do I care? maybe not). Very over-the-top, but way too much fun and a very cool flex.
